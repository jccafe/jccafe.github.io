Version 2.0 • 國中自然會考 50 題結構化大數據自學系統

1 系統概述與核心價值

本 V2.0 系統特別針對「115會考自然科試卷（115P_Nature.pdf）」的 50 題特徵，結合國中自然科學 108 領綱（1613715832381Ld8uk4KU.pdf）設計而成。我們在純前端技術框架中，精準封裝了 50 題結構化數據。系統在設計上徹底摒棄了可能被校園內網、安全 sandbox 或 iframe 阻擋之外部大框架，運用**原生動態 SVG 與 CSS 演算**繪製圖表，並搭載 KaTeX 渲染美化數理公式，保證 100% 開啟與操作不卡頓。

2 多維度篩選與 LocalStorage 資料流設計

系統進度追蹤模組利用瀏覽器 localStorage，在前端進行實時的進度保存。資料流邏輯如下：

1. 學生點選題目，勾選「標記為已複習」 ➔ 觸存事件 ➔ localStorage.setItem('115_nature_v2_completed', JSON.stringify(completedSet)) ➔ 調用 updateProgressGauges() 重新渲染頂部進度條與 SVG 學科分布圓環。 2. 學生在「小試身手挑戰」選擇答案 ➔ 判斷對錯 ➔ 若為首次作答，存入 localStorage.setItem('115_nature_v2_scores', JSON.stringify(scoreMap)) ➔ 調用 updateProgressGauges() 更新整體答對率計量。

3 50 題全量 JSON 數據庫規格定義 (JSON Schema)

系統每道題目的儲存結構都定義在前端變數 questionDatabase 中，資料結構如下：

{ "id": "115_N_01", "number": 1, "subject": "地球科學", "book": 5, "chapter": "宇宙與天體", "concept": "光年與距離單位", "curriculumCode": "ah-Ⅳ-2", "type": "知識記憶", "difficulty": "易", "brief": "科學家發現距離地球10光年的恆星...", "detailedExplanation": "光年為「距離」單位而非時間...", "commonMisconception": "易將光年誤認成時間單位，或以為要乘上光速計算...", "quiz": { "text": "此時天體大爆炸，地球上最快何時觀測到？", "options": [ { "text": "2036年", "isCorrect": true, "desc": "正解..." } ] } }

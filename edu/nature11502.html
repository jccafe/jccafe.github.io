1 系統核心價值與建置目的

本系統專為 115 年會考自然科之高畫質結構化剖析而建置。自然科試卷（115P_Nature.pdf）呈現出高密度的圖表判讀與實驗探究趨勢。為了提供全台國中學子在沙盒環境（無外部網路、高安全性阻隔、校園網路防火牆）中，仍能 100% 流暢無阻地使用，本系統將 108 自然領綱（1613715832381Ld8uk4KU.pdf）的學習表現與核心試題緊密黏合。採用純 HTML5/CSS3/JS 無外部 JS 函式庫相依技術，使複習資源即刻載入。

2 軟體工程與高安全性架構 (Architecture & Security)

無依賴本地渲染設計

一般前端框架常引入 Chart.js、Google Charts 等外部 CDN。然而在許多校園或 iframe 安全沙盒下，外部 Script 會遭到 CSP（內容安全策略）封鎖，導致圖表遺失、頁面報錯。本系統完全使用**原生 DOM 操作與 SVG 動態流式圖表**，無須載入任何外部大圖庫。

自適應防破版佈局

針對 iPad 及手機等不同螢幕尺寸，左清單、右詳情在桌面端為雙欄，並將右側設定為 sticky top-24。在小於 1024px 的螢幕則自動降為單欄垂直，確保完全不產生橫向滾動。

3 試題結構 JSON 綱要 (Database Data Schema)

系統所載的 50 題底層 JSON 資料庫完全落實以下數據模型。以保證其易於擴增至未來的 116 年會考考卷分析：
